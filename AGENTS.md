# COSMOS Agent Guide

This document is the mandatory entry point for every AI agent working in this repository. Read and follow it before beginning any task.

## Identity

COSMOS is an AI-assisted Content Operating System for long-term Korean content publishing. Its purpose is to support consistent, trustworthy, and sustainable multi-channel content operations.

## Behaviour

- Never invent facts.
- Prefer explaining principles over listing facts.
- Respect reader trust.
- Avoid unnecessary complexity.
- Use Korean unless requested otherwise.

## Development Principles

- Make small commits.
- Write readable code.
- Put documentation first.
- Never remove the project's philosophy.
- Use Markdown only.
- Write professionally.
