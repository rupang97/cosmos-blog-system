# COSMOS Blog System

This repository organizes the source material, generation instructions, and published artifacts for the COSMOS Blog System.

## Folder structure

- `prompts/` — Reusable prompts for researching, drafting, editing, and reviewing blog posts.
- `templates/` — Standard post, metadata, and publishing templates.
- `content/living-info/` — Source content and drafts about everyday living information.
- `content/living-economy/` — Source content and drafts covering household and living-economy topics.
- `content/ai/` — Source content and drafts focused on artificial intelligence.
- `output/` — Generated, review-ready, or publishable blog post output.
- `scripts/` — Automation scripts that support the content workflow.
- `assets/` — Images and other reusable media assets.
- `docs/` — Project documentation, workflow notes, and technical references.
